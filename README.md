## ⏰ Function Description

### ⏰ To-Do List & 루틴 추천
<img width="2400" height="1350" alt="image" src="https://github.com/user-attachments/assets/599a5e57-58a0-4cf4-b840-04ac01ecf3ed" />

- To-Do List에 할 일 추가 가능  
- 완료 시 마일리지 지급  
- 루틴 추천은 3가지 세트 제공, 재조합 시 무작위 재추천  
- 추천 루틴을 To-Do List로 바로 추가 가능  

---

### ⏰ 알람 기능
<img width="2400" height="1350" alt="image" src="https://github.com/user-attachments/assets/188a7a27-3b13-48dc-b4d8-9373a1b604b8" />

- 시/분/초 단위 설정  
- 알람 저장, 삭제, 전체 삭제  
- 알람 시간 도달 시 소리 재생 + 퀴즈 발생  
- 퀴즈 오답 시 창 닫기 불가  

---

### ⏰ 캘린더 기능
<img width="2400" height="1350" alt="image" src="https://github.com/user-attachments/assets/031ec134-7714-4960-b315-572fcc22423b" />

- 전체 월력 표시  
- 다음달/이전달 이동  
- 일정 입력 기능:
  - 날짜 클릭 → 월/일/년도/시간/일정 이름 입력 후 추가/삭제  
- 가계부 기능:
  - 날짜 클릭 → 월/일 설정 후 수입/지출 기록 및 되돌리기  
  - 월 단위 총 금액 표시  
- 칼로리 기록 기능:
  - 날짜 클릭 → 월/일 설정 후 칼로리 기록 및 되돌리기  
  - 월 단위 총 칼로리 표시

### ⏰ 시간표 기능
